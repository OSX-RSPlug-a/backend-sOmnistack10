# backend-sOmnistack10

Projeto DevRadar, dos aprendizados da semana OmniStack 10.0;


Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

      Nodejs;
            Express;
            Axios;
      Mongodb;


To init this project:

      - npm i (to push and install all dependencies)

      - npm run dev

      - npm start

      localhost:3333
